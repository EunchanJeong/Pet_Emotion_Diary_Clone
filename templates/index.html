STEP 1 : 반려동물과 찍은 동영상 업로드

✅ 일기 작성 시작

가장 먼저 상단바의 Diary 탭을 클릭해주세요.
만약 등록된 반려동물이 없다면, 반려동물 등록을 진행해주세요.
일기목록 화면의 오른쪽 상단에 있는 일기쓰기 버튼을 눌러주세요.

😊 비디오 업로드

일기를 생성하고 싶은 순간이 담긴 동영상을 선택해주세요.
선택한 동영상을 왼쪽의 비디오 업로드 섹션의 파일 선택을 통해 업로드 해주세요.

STEP 2 : 업로드된 영상을 바탕으로 AI 일기 생성

✅ AI로 일기 자동 생성

동영상을 업로드하게 되면, 인공지능 모델이 가장 먼저 반려동물의 행동과 감정을 예측합니다.
동시에 기록된 반려동물의 성격도 반영하여 반려동물 시점에서의 일기를 자동 생성합니다.
일기에 꼭 기록하고 싶은 내용이 있다면 해당 내용을 추가할 수 있습니다.

STEP 3 : 반려동물과의 추억을 자유롭게 꺼내보기

💝 반려동물 일기 모아보기

일기 목록에서 달력/리스트 형식으로 일기를 마음껏 감상하세요.
북마크 기능을 통해 다시 보고 싶은 일기를 표시할 수 있습니다.
